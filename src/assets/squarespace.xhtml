Squarespace XML namespace

The namespace name http://www.lvier.de/assets/squarespace.xhtml is used for the proprietary tags of the Squarespace CMS. Squarespaces own documentation is sadly sorely lacking, so an actual authoritative source for the correct use of this namespace is nonexistent. Use with caution, as the schema for this namespace is derived from the documentation available and may as such be faulty as well as incomplete.

The schema corresponding to this namespace can be found at http://www.lvier.de/assets/squarespace.xsd.
